About This Project

This project is built for exploring and visualising public safety fine data using D3.js. It demonstrates how seatbelt-related fines are distributed across jurisdictions and age groups in 2023.

Technologies used include:

  • D3.js (Data-Driven Documents)
  • Bootstrap 4 (for base layout)
  • Custom CSS (for layout & styling)

Built as part of an academic project to showcase interactive data visualisation dashboards.
